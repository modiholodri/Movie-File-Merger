Bugs & Ideas

If you find a bug, have suggesetions or ideas concerning MFM or any of the books, please use one of the below listed issue trackers to submitt your suggestion.

MFM is basically built up out of 3 separate GitHub projects. That's why it has also 3 bug and suggestions trackers.

  • The Movie File Merger Bug Tracker, where you can log bugs and suggestions related to the MFM program and this web site.
  • The Movie File Merger User Manual Bug Tracker, where you can log bugs and suggestions related to the MFM user manual GitBook.
  • The How to Organize Videos on Your Hard Disks Bug Tracker, where you can log bugs and suggestions related the how to organize videos GitBook.
